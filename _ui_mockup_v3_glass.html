  • ★ 이중 글라스 패널 구조 — 사이드바/콘텐츠가 각각 독립 글라스 레이어. 사이드바는 높은 투명도(배경 비침), 콘텐츠는 낮은 투명도(가독성 확보)

  • ★ 윈도우 코너 24px — Tahoe 기본 ~24px 반영. 사이드바/콘텐츠도 매칭 라운드
  • ★ 이모지 → SVG stroke 아이콘 (SF Symbols 스타일). 시스템 폴더만 컬러 컨테이너 유지
  • ★ 글라스 엣지 하이라이트 — 사이드바 상단/우측 경계에 빛줄기 (::before, ::after)
  • ★ 검색 축소형 — 아이콘만 표시, 클릭 시 확장 (Tahoe 방식)
  • △ Pill 배경 더 미묘하게: 0.04 → 0.025
  • △ 파일 행 높이 컴팩트: 30px → 26px
  • △ 툴바 높이 48 → 44px, 하단 패스바 32 → 30px
